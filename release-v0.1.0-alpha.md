# Agent Mesh Mission Control v0.1.0-alpha

First public alpha of the rebuilt dashboard.

## Highlights

- mesh-first real-time operator dashboard
- permanent agent dock for Lead, Hermes, and IrisEye
- clickable ops strip, alerts, and utility controls
- memory summary, memory events, cause ranking, and routing impact
- live screenshots captured from the running app

## Stack

- React 19 + Vite frontend
- FastAPI + WebSockets backend
- MLX / local memory-routing operator focus

## Notes

- local experimental wrapper scripts are intentionally not part of the committed public product surface
- this alpha is focused on the dashboard, backend polling, memory/routing state, and operator UX
